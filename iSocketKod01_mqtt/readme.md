## Ujikaji 1b: kawal LED - ON dan OFF
```
  digitalWrite(relay01,HIGH);
  delay(1000);
  digitalWrite(relay01,LOW);
  delay(1000);
```

```
  digitalWrite(relay01,HIGH);
  digitalWrite(relay02,HIGH);
  delay(1000);
  digitalWrite(relay01,LOW);
  digitalWrite(relay02,LOW);
  delay(1000);
```
